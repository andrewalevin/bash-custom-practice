## Alias bb - BBEdit

**Add to ~/.profile**

```bash
nano ~/.profile
```

**This Code**

```bash
alias bb="open -a /Applications/BBEdit.app"
```

**And make to install at once**

```bash
source .profile
```

**How to use**

```bash
bb script.sh
```
